Код написаний на Java.
Файл для зчитування набору фрагментів має буди розміщений у тій же директорії, що й код програми для вирішення задачі.
Для цього доступний файл source.txt у комплекті, однак можна використати інший текстовий файл. На початку роботи програма запитує ім'я файлу для завантаження фрагментів.


Для запуску програми доступні два варіанти.

1. Якщо наявна JVM, то можна виконати компіляцію та запуск з консолі.
 - Скомпілюйте код командою 
	javac DigitalPuzzle.java
   у консолі (cmd) у папці, де розміщено файл з розширенням .java
 - Запустіть код за допомогою команди 
	java DigitalPuzzle
   у консолі (cmd) у папці, де розміщено файл з розширенням .java

2. Якщо JVM відсутня, можна скористатись Docker. У папці наявний відповідний Dockerfile для створення образу. Для запуску виконайте наступне:
 - У консолі в директорії, де розміщено файли виконайте команду створення образу:
	docker build -t digital-puzzle .
 - Запустіть контейнер з доступом до командної строки командою:
	docker run -it digital-puzzle

В обох випадках введіть назву файлу з фрагментами у cmd та зачекайте на результат.

*Налаштування запуску зроблені за допомогою AI Copilot.
